# 安装 linux kernel headers

```shell
sudo apt-get install linux-kernel-$(uname -r)
```

# 安装网卡驱动

首先用 lshw 命令查询网卡的型号

```shell
lshw -class network
```

然后在产品的官网下载编译安装

https://www.realtek.com/en/directly-download?downloadid=c4676ea8d1adc3c6f28a4ae2c3e2787a
